q: 如何关闭html页面在ios下的键盘首字母自动大写?
```js
autocapitalize="off"
```
```html
<input type="text" autocapitalize="off" />
```
tips
仅支持移动端，Firefox for Android 暂不支持

q: css中哪个属性会影响DOM读取文档流的顺序
```js
writing-mode
```
```html
<div class="vertical-middle"> 
  <h3>春晓</h3>
  <p>
    春眠不觉晓
    <br />
    处处闻啼鸟
    <br />
    夜来风雨声
    <br />
    花落知多少
    <br />
  </p>
</div>
```
```css
.vertical-middle {
  writing-mode: vertical-rl;
}
```
