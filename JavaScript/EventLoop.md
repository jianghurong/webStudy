## intro
Event Loop是计算机系统中的一种运行机制
javaScript采用这种机制来处理单线程的一些问题
javaScript是一种单线程语言
EventLoop: 在程序中设置两个线程，一个负责程序本身的运行，另一个负责主线程与其他进程的通信，被称为"Event Loop"线程
