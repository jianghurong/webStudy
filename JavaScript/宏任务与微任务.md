### 优先级
微任务>宏任务（异步任务）

### 分类
宏任务: setTimeout, script, setInterval, then
微任务: promise

### 实例
```js
setTimeout(function () {
    console.log(1);
}, 0);
new Promise(function (a, b) {
    console.log(2);
    for (var i = 0; i < 10; i++) {
        i == 9 && a();
    }
    console.log(3)
}).then(function () {
    console.log(4)
});
console.log(5);
```
